### .gitignore 구성

#### Go 관련 제외 항목
- 바이너리 파일: `aicli`, `*.exe`
- 테스트 커버리지: `*.out`, `coverage.html`
- 의존성 캐시: `vendor/` (사용 시)

#### 프로젝트 특화 제외
- 빌드 아티팩트: `dist/`, `build/`
- 로컬 설정: `.env`, `config.local.yml`
- 임시 파일: `*.tmp`, `*.log`

#### IDE/에디터 제외
- VS Code: `.vscode/` (설정 파일 제외)
- GoLand: `.idea/`
- Vim: `*.swp`, `*.swo`
